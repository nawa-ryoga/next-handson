## 基本情報
React・Next.js・TypeScriptの学習を行うためのリポジトリです。
下記ページを参考にして同様のランキングを表示するページを作っています。

https://pixiin.com/ranking-japan-boxoffice2022/

## 環境
- Next.js 13
- React 18
- TypeScript 4.9
- Docker
- Docker Compose

## セットアップ
```bash
# docker compose up -d
make up 

# docker compose exec app bash
make bash

npm install
npm run dev
```
